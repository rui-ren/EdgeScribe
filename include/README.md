# Vendored dependencies

This directory contains single-header libraries downloaded at build time:

- **miniaudio.h** — Cross-platform audio capture/playback (Public Domain / MIT-0)
  - Source: https://github.com/mackron/miniaudio
  - Download: `curl -sL -o miniaudio.h https://raw.githubusercontent.com/mackron/miniaudio/master/miniaudio.h`

These files are in `.gitignore` and downloaded during build or CI.
