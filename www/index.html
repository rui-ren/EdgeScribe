Download AI Models

EDGESCRIBE needs AI models to work. Open a terminal and run these commands to download them:

edgescribe pull nemotron Speech Recognition (~670 MB)
edgescribe pull qwen3-vl Chat & Image Analysis (~1.5 GB)
edgescribe pull kokoro Read Aloud (~300 MB)
